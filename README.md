# 项目名称
基于大模型的保险条款通俗化解读与对比系统

# 项目简介
本项目旨在构建一个基于大语言模型（LLM）的保险条款智能解读与对比工具，实现：
- 将复杂保险条款转化为通俗易懂的语言解释
- 自动抽取关键字段（保额、免责条款、等待期等）
- 支持不同产品之间的结构化对比
- 基于检索增强生成（RAG）提供问答与原文依据支持

# 示例保险文件

从[中国保险行业协会下载](https://www.iachina.cn/col/col94/index.html)的示例文件：

# 系统架构
系统采用“双层架构”设计：

##  1. 结构化主线（核心流程）
```text
用户上传条款
↓
文本清洗
↓
基于 schema 的信息抽取（LLM → JSON）
↓
结构化存储
↓
字段级解释 / 产品对比
```
作用：
- 产品对比
- 通俗化解读
- 可扩展分析

说明：
结构化层为系统核心，负责通俗化解读与产品对比功能。

## 2. RAG 辅助线（增强能力）
```text
条款原文
↓
文本切块
↓
Embedding 向量化
↓
向量数据库存储
```
作用：
- 自由问答
- 原文依据支持
- 降低模型幻觉风险

说明：
RAG 不参与结构化抽取，仅在问答或复杂解释时调用。
